TinCar

Problematica a resolver

Sabemos que el trafico de Bogota es poco fluido, esto se debe a diferentes factores, uno de estos factores y el cual nuestro
proyecto va enfocado es la invacion a la via publica por vehiculos mal parqeuados.

¿Como se resolvera?

Por medio de un software que implementaremos en una aplicacion movil llamada TinCar.

Descripcion y Objetivos del proyecto

  • Descripcion

    El proyecto TinCar pretende proporcionar espacios seguros de aparcamiento a los conductores de diferentes tipos de vehículos, y más concretamente se espera lograr un tráfico más fluido y armonioso en diferentes zonas de la ciudad de Bogotá.

  • Objetivo de negocio

    Los objetivos comerciales de este proyecto son el apoyo directo de nuestro plan estratégico para hacer uso de parqueaderos compartidos y reducir la congestión vehicular por causa del mal parqueo. de TC y reducir los costos asociados con pérdidas y desperdicios.

  • Proposito

    El proyecto TinCar nace ante la necesidad de mejorar la movilidad ante la presencia de innumerables trancones ocasionados por actores viales los cuales se suelen movilizar mayormente en vehículos como carros y motos. Los inversionistas recuperaran su dinero gracias a que nuestro software proporciona ganancias al buscar y encontrar garaje para los vehículos que lo están buscando

Tecnologías Utilizadas

Para el desarrollo y ejecucion de nuestra a plicacion y pagina web tendremos en cuenta:

  • Desarrollo nativo: se crean apps con lenguajes específicos para sistemas operativos móviles como Android o iOS. Por ejemplo, se puede usar Swift para iOS o Kotlin para Android.
  • Desarrollo híbrido: se crean apps que se pueden ejecutar en múltiples plataformas con un solo código fuente. Por ejemplo, se puede usar Ionic, Flutter, React Native o Xamarin
  • Desarrollo web progresivo: se crean apps que se comportan como una app móvil, pero se acceden desde un navegador web. Por ejemplo, se puede usar HTML, CSS y JavaScript
